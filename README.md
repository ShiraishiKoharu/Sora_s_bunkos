# 📚 Sora's Bunkos

A bunko-style e-book project featuring stories by **Akihiko Sora**.

> _"Những câu chuyện nhẹ nhàng về tình yêu và tuổi trẻ."_

---

## 🌸 About This Project

This project transforms original stories into beautifully typeset bunko-format PDFs, available in three editions:

| Edition       | Theme | Description                                   |
| ------------- | ----- | --------------------------------------------- |
| 📖 **Normal** | Gray  | Minimalist design                             |
| 🌸 **Sakura** | Pink  | Limited edition with cherry blossom ornaments |
| ❄️ **Winter** | Blue  | Special edition dedicated to 美幸             |
